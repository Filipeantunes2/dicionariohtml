Tag de definição de hiperlink

A tag <a> define um hiperlink, que é usado para vincular de uma pagina a outra.

Dica:

Uma pagina vinculada normalmente é exibida na janela atual do navegador, e menos que você especifique outro destino

Exemplo:

Visite meu site!

Tag address

A tag <address> define as informações de contato do autor/proprietário de um documento ou artigo. As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, conta de mídia social, etc.

Exemplo:

Escrito por Filipe Antunes.

Tag de áudio

A tag de áudio é usada para incorporar conteúdo sonoro em um documento, como música ou outros fluxos de áudio. A tag contém uma ou mais tags com diferentes fontes de áudio. O navegador escolherá a primeira fonte que ele suporta. O texto entre as tags e só será exibido em navegadores que não suportam o elemento. Há três formatos de áudio suportados em HTML: MP3, WAV e OGG.

Tag de vídeo

A <video>tag é usada para incorporar conteúdo de vídeo em um documento, como um clipe de filme ou outros fluxos de vídeo. A <video> tag contém uma ou mais <source> tags com diferentes fontes de vídeo. O navegador escolherá a primeira fonte que ele suporta.

Tag detalhes

A tag <details> especifica detalhes adicionais que o usuário pode abrir e fechar sob demanda. A tag <details> é frequentemente usada para criar um widget interativo que o usuário pode abrir e fechar. Por padrão, o widget é fechado. Quando aberto, ele se expande e exibe o conteúdo dentro dele.

Exemplo:

Dica

A tag <summary> é usada em conjunto com <details> para especificar um título visível para os detalhes.

Tag input element

A tag <input> especifica um campo de entrada onde o usuário pode inserir dados. O <input> elemento é o elemento de formulário mais importante. O <input>elemento pode ser exibido de várias maneiras, dependendo do atributo type.

Os diferentes tipos de entrada são os seguintes:
